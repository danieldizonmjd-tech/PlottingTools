## Data context used by these plots

- Main data file: `paper_data_files/apjsadc397t2_mrt.txt`
- Project reports state a full sample of 20,654 YSOs with strong class imbalance:
  - ClassII is the majority class
  - NV is the majority LC type
- Variability bins are based on `delW2mag`:
  - Low: `< 0.2`
  - Medium: `0.2 to < 0.5`
  - High: `>= 0.5`

## How to read these plots correctly

- Row-normalized heatmaps:
  - Compare values across columns within a row.
  - Rows sum to 100 percent.
- Residual heatmaps:
  - Positive values mean overrepresentation.
  - Negative values mean underrepresentation.
  - Absolute residual around 2 or larger is typically notable.
- Correlation heatmap of numeric metrics:
  - Shows linear association among metrics.
  - It does not prove causation.
- Scatter plots:
  - Check population trends and outliers.
  - Always consider brightness-dependent noise and selection effects.

## Conclusions

These conclusions combine the code design and the included project reports.

1. Variability amplitude is not random across morphology classes.
Light-curve type carries meaningful information about amplitude regime, with irregular and eruptive shapes tending to stronger variability than NV.

2. Evolutionary class has a real but weaker relationship with variability than morphology does.
This fits a picture where class influences environment and disk state, but short-timescale light-curve behavior is more directly linked to immediate physical processes.

3. Color behavior adds physical discrimination within irregular sources.
Redder-when-fainter trends are compatible with extinction changes, while blueing during brightening is compatible with accretion-related processes.

4. The sample is strongly imbalanced.
Any conclusion for tiny classes such as `uncertain` and rare LC types should be treated as provisional because small-number statistics can dominate those cells.

5. The strongest workflow is:
use normalized heatmaps and residual maps for categorical interpretation,
then use numeric-metric and color-slope scatter plots for physical context.
